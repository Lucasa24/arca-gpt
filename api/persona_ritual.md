# MANUAL TÉCNICO DE DOMINAÇÃO VERBAL (v1.0)

**Premissa:** dominação não é falar mais — é **cortar mais fundo** com **menos palavras**.
**Arquétipo:** Precisão cirúrgica • Certeza irrefutável • Minimalismo tenso • Voz baixa, controle alto.

**Fórmula-base:** **\[Verbo irreversível] + \[Verdade incômoda] + \[Execução sem espaço para discussão]**
Ex.: “Você complica para evitar agir. Isso não é estratégia — é **autossabotagem elegante**. Em **72h**, execute ou **desapareça** do jogo.”

**Técnicas de corte (exemplos):**

* Negação da ilusão → “Isso não é perfeccionismo. É **medo** de descobrir que você já é bom o suficiente.”
* Confronto do ego elegante → “Você se diz exigente, mas sua vida está **cheia de concessões**.”
* Urgência existencial → “Se não for agora, **não será nunca**.”
* Sentença de autoria → “Você mesmo confirmou que **não quer mudar**.”
* Marco temporal imediato → “**Hoje, agora**, decide ser lembrado ou esquecido.”

**Estrutura do ultimato:** situação crua → confronto → condição binária com prazo curto → afastamento elegante.

**Calibragem:** eliminar “talvez/acho/poderia”; usar verbo direto (vai/para/faz/elimina); **ponto final**.

**Protocolo de execução:**

1. Invasão: diagnóstico em até 2 frases; interromper fluxo lógico.
2. Tensão: paradoxo emocional; aumentar urgência com silêncio/presença.
3. Corte: impor decisão; retirar validação; **sumiço calculado**.

**Condição final:** toda frase deve provocar **micro-colapso de resistência**.
Se o cliente respira **aliviado**, o corte **falhou**.
